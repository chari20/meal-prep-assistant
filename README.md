# 🥗 AI Meal Prep Assistant

A full-stack web app that uses the Claude AI API to generate personalized meal prep plans based on your nutrition goals and available ingredients.

## Live Demo
*Coming soon*

## Features
- Enter your calorie and protein goals
- Specify food preferences and dislikes
- List available ingredients
- Get an AI-generated one-day meal plan with macros, prep instructions, and a grocery list

## Tech Stack
- **Frontend:** React (Vite)
- **Backend:** Node.js / Express
- **AI:** Anthropic Claude API
- **Version Control:** Git / GitHub

## Getting Started

### Prerequisites
- Node.js v20+
- An Anthropic API key ([get one here](https://console.anthropic.com))

### Installation

1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/meal-prep-assistant.git
cd meal-prep-assistant
```

2. Install server dependencies
```bash
cd server
npm install
```

3. Install client dependencies
```bash
cd ../client
npm install
```

4. Create a `.env` file inside the `server/` folder
